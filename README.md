# occhat

Проект выложен по адресу https://vlad-lis.github.io/occhat/

## Запуск проекта

1. Клонировать репозиторий:

```
git@github.com:vlad-lis/occhat.git
```

2. Установить зависимости:

```
npm i
```

3. Запустить проект:

```
npm run dev
```

Проект запустится локально на http://localhost:5173/occhat/
